# Aniket Awasthi — Portfolio

Personal portfolio built with **Vite + React + TypeScript + Tailwind CSS**. Showcases the projects I've shipped, my work experience, and the stack I use day-to-day.

Live: [aniket-portfolio-murex.vercel.app](https://aniket-portfolio-murex.vercel.app/)

---

## Stack

- **React.js** + **TypeScript** — component-driven UI
- **Tailwind CSS** — utility-first styling
- **Framer Motion** — animations
- **Vite** — build tooling
- Deployed on **Vercel**

## Running locally

```bash
git clone https://github.com/Ai983/Aniket-Portfolio-main
cd Aniket-Portfolio-main
npm install
npm run dev
```

Open `http://localhost:5173`.

## Project structure

```
components/     # Hero, About, Skills, Experience, Projects, Education, Contact
constants.tsx   # All content lives here — edit this to update copy/data
types.ts        # TypeScript interfaces
App.tsx         # Root layout
```

All text content (projects, experience, skills, etc.) is centralized in `constants.tsx` — no need to dig through components to update copy.
